QR Attendance Frontend
This app is designed to make classroom attendance easier and quicker. Instructors can create, modify, and view their classroom attendance along with graphical statistics. A series of code protected QR codes are generated and displayed for students to be marked present by scanning the QR code with the camera on their smartphone. Since this is a current work in progress with the aim to release for web, iOS and Andorid only sample code is open to the public.
